¿Qué puedo hacer aquí?

  • Un espacio solo para ti — WordPress es la base de gran parte de la web, pero aquí es solo para ti. Úsalo como cuaderno de notas, espacio para proyectos o lugar para experimentar con tu creatividad; no hace falta que publiques nada.
  • Totalmente personalizable — Cambia el tema y organízalo todo como más te guste. Aprenderás a usar WordPress de forma natural a medida que avances.

Cosas que debes saber

  • Está ligado a este navegador — Tu Playground se almacena en la memoria de este navegador, por lo que no puedes acceder a él desde otros dispositivos. Esto lo convierte en un espacio muy privado.
  • ¿Listo para dar un paso más? — Traslada tu Playground a un hospedaje dedicado de WordPress para acceder a él desde cualquier dispositivo, compartirlo públicamente o invitar a personas concretas a colaborar.

El menú superior

Busca el icono de la cuadrícula () en la barra superior. Desde aquí puedes:

  • Instalar aplicaciones — Hemos seleccionado algunos plugins que funcionan bien para uso privado. También encontrarás allí el directorio completo de plugins de WordPress, por si quieres echarles un vistazo.
  • Empezar de cero — Restablece tu WordPress o visita playground.wordpress.net para realizar experimentos rápidos y temporales
